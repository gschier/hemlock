## About Hemlock

Hemlock is a web application framework for [Go](https://golang.org/).

## Installation

```bash
go get github.com/gschier/hemlock
```

## Getting Started

Create a new application in desired folder

```bash
hemlock new path/to/my/app
```

Build and run the sample app

```bash
cd path/to/my/app
hemlock serve --watch
```
